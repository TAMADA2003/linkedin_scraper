parsing_profile.py

1. Данные для перехода по профилям, мы забираем из файла "tsp_profiles.csv"
2. Обработаные профили, мы ложим в файл "scrapped_profiles.csv"
3. Команды для запуска скрапера "python parsing_profile.py --id 242 --login TestAOGTalent@gmail.com --password 55217966Alex2015 --delay 0.4"
"parsing_profile.py" = файл от куда происходит запуск 
"--id "  = указивает с которого id мы начинаем запуск обработки профилей, id мы берем из файла "tsp_profiles.csv"
"--login " = указывает логин для авторизации на Linkedin
"--password " = указывает пароль для авторизации на Linkedin
"--delay' = узаывает задежку между переходами по профилям (по стандарту 0.4)

	Данный проект работает на версии "Python 3.11.3"
